

# 版本管理
## 为什么我们需要版本管理


### 优点
1. 改善版本号问题，减少沟通成本
2. 可查看或者回复旧版本

### `Git` ---- 分布式版本管理工具

#### 本地版本控制管理工具的缺点
- 无法团队协作
#### 集中式版本控制管理工具的缺点
- 可团队协作
- 依赖中央服务器
#### 分布式版本控制管理工具的缺点
- 可团队协作
- 每个客户端都可以保留最新版本的快照
- 在大团队开发中，可以分别和不同工作小组的人相互协作
